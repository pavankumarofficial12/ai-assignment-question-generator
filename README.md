# AI Assignment Question Generator

An intelligent AI-powered system that **automatically generates personalized assignment questions** based on a student's performance, grade, feedback, and course topics. Powered by **AWS Bedrock (Llama 3)**.

---

## ✨ Features

- Generates 2–3 thoughtful, level-appropriate questions
- Analyzes student performance (Grade, Score, Feedback)
- Adapts difficulty: Basic → Intermediate → Advanced
- Personalized based on Main Topic & Sub Topic
- Clean and structured JSON response
- FastAPI-based REST API

---

## 🛠 Tech Stack

- **Backend**: FastAPI
- **AI Model**: AWS Bedrock (Llama 3 70B)
- **HTTP Client**: httpx
- **Environment Management**: python-dotenv
- **AWS SDK**: boto3
